# ¿ Que es QuickAdventure ?
QuickAdventure es una aplicación desarrollada en AndroidStudio mediante Java nativo y Firebase, la cual busca un plan de ocio personalizado según tu localización y gustos preseleccionados antes de buscar plan.

Ideal para ser turista.
