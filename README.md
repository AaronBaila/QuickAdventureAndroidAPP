# ¿ Que es QuickAdventure ?
QuickAdventure es una aplicación desarrollada en Android Studio utilizando Java nativo y Firebase. Su objetivo es ofrecer planes de ocio personalizados según tu localización y tus gustos, previamente seleccionados antes de realizar la búsqueda.
Es ideal para turistas que desean descubrir actividades adaptadas a sus preferencias.

# Estado del proyecto:
Actualmente, la aplicación cuenta con funcionalidad de login y búsqueda de planes según la localización del usuario. No obstante, todavía queda mucho por mejorar y actualizar.
Se trata de un proyecto que inicié como parte de mi aprendizaje durante el Grado Superior y que posteriormente dejé en pausa. Mi intención es retomarlo próximamente con más conocimientos y tiempo para seguir desarrollándolo y mejorándolo.
